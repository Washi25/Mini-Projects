# Mini-Projects
In this repo I will be documenting my mini projects. 

The project ideas are generated using AI, I used ChatGPT to generate project briefs without any code examples.
The use of AI-generated project briefs forces me to think like an engineer/analyst, it also allows me to improve my problem-solving skills and avoid copy-paste dependency which a recurring issues nowadays with recent graduates. It also teaches me on how to build portfolio-quality understanding and helps you learning system design and project structure. All of these skills are beneficial for future roles.
